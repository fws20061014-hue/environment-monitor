# 环境数据记录平台

这是一个用于展示 STM32F407ZGT6 云端环境数据的静态网页原型。

当前功能：

- 工人体温、附近粉尘、摔倒、头盔报警与电量状态看板
- 异常工人优先展示和全部工人状态管理
- 模拟施工地图工人定位页面，可选切换百度地图 API
- 工人身份信息及图片上传
- 可选云端反馈接收后端，见 `server/`

目前数据来自前端模拟函数。后续云端接口确定后，替换 `app.js` 中的 `fetchCloudData()` 即可接入真实数据。

## 居民反馈云端接收

前端通过 `config.js` 读取反馈后端地址：

```js
window.ENV_MONITOR_CONFIG = {
  feedbackApiBase: "https://你的反馈后端地址",
  baiduMapAk: "你的百度地图浏览器端AK",
  siteCenter: {
    lng: 116.404,
    lat: 39.915,
  },
};
```

定位页面默认使用模拟施工地图，不依赖外部地图服务即可演示工人位置和风险状态。需要切换真实地图时，可填写百度地图开放平台申请的浏览器端 JavaScript API AK，并按部署地址配置 Referer 白名单。

后端代码在 `server/` 文件夹中。部署后，居民提交会保存到后端；如果后端地址为空或临时不可用，网页会自动改为本机暂存，方便演示。
